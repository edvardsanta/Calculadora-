# Calculator

## Tools used
- ** Vite **
- ** yarn **
- ** React **
- ** Bootstrap **
- ** React-Bootstrap **
- ** Termux **

## Getting started
You can calculate any elementary math operation
(Sum, Subtraction, Division and Multiplication).

## Using Calculator
- `git clone https://github.com/edvardsanta/Calculator_react`
- `cd Calculator_react`
- `yarn dev or npm rum dev`
- `open <http://localhost:3000>`
